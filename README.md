## Install

```bash
git clone git@github.com:jiacai2050/wiki.git
git clone git@github.com:jiacai2050/wiki.git wiki-book
cd wiki-book
git checkout gh-pages

npm i -g gitbook-cli
gitbook install

gitbook serve
```
